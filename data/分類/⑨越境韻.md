# ⑨越境韻
_軸: **構造軸**_

> 英語と日本語を跨いで踏む韻。同じ母音骨格で言語を越える。

> **わかりやすく言うと**：「英語の単語」と「日本語の単語」が同じ音で着地する韻。例：「ストーリー (story)」が日本語の「過ごし/素通り」と同じ -ō-i 音で揃う。言語の壁を越える現代日本語ラップの特技。

## 章の本質

異なる言語の音が同じ韻骨格を共有するという、現代日本語ラップに特有の最先端技法。英単語が日本語の音韻体系に取り込まれ、その末尾モーラの母音が日本語句と一致することで成立する。

### 仕組み

- 英語の単語は日本語話者の耳に a/i/u/e/o の5母音体系で再解釈される
- 「dance」→ ダンス (da-N-su) → 末尾 `-u`
- 「style」→ スタイル (su-ta-i-ru) → 末尾 `-i-u`
- これらを日本語の漢語・和語と同じ末尾母音で着地させることで、言語が違っても韻が成立する

### 越境韻が成立する条件

「行のどこかに英語が散りばめられている」だけでは越境韻ではない。
末尾（韻の着地点）に英語の語が来る行と、日本語の語が来る行が、同じ韻骨格を共有していることが必須条件。

```
越境韻ではない:
  Tictoc はじく電卓もなく      ← 末尾「もなく」(日本語)
  Nature Boy になったような感覚 ← 末尾「感覚」(日本語)
  (英語は冒頭にあるだけ、韻には参加していない)

越境韻として成立:
  夢見てた 錆びた鉄格子        ← 鉄格子 (-gō-shi)
  絶望して 涙で過ごし          ← 過ごし (-go-shi)
  救いはねぇ 神だって素通り    ← 素通り (-dō-ri)
  なら自分の手で書き換えるストーリー  ← ストーリー (story / -tō-rī)
```

→ 借用語「ストーリー (story)」が、日本語の漢語/和語の末尾と すべて -ō-i (オー音 + イ音) で着地。
→ 英語が日本語音韻体系に取り込まれ、native な末尾と同じ振動を生む典型的な越境韻。

### 聴感的効果

- **越境感**: 言語の壁を破る、グローバル感
- **コードスイッチの興奮**: 英語が入る・戻る切替がリズム
- **アイデンティティ**: 「日本語ラップだが英語も内在化している」音楽的記号

### 文化的意義

BUDDHA BRAND、スチャダラパー、Awich、BAD HOP、JP THE WAVY、Creepy Nuts など、日本語ラップ史を通じて英語混在は標準装備であり続けてきた。越境韻は単なる技法ではなく、音楽ジャンルとしての所属を示す記号。

### 注意

英単語がたまたま入っているだけでは越境韻ではない。英日の語が同じ末尾母音で着地していることが必須条件。
