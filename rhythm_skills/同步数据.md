# 同步数据

将本地 Obsidian 韻辞典 vault（`/Users/koishi810/Desktop/韻辞典/`）的最新内容同步到 Web 项目的 `data/` 目录。

## 步骤

1. 用 `diff` 或文件修改时间比较以下三个目录，找出有变动的文件：
   - `~/Desktop/韻辞典/曲/` → `data/曲/`
   - `~/Desktop/韻辞典/分類/` → `data/分類/`
   - `~/Desktop/韻辞典/アーティスト/` → `data/アーティスト/`

2. 列出新增、修改、删除的文件清单，让用户确认后再执行同步。

3. 用户确认后执行：
   ```bash
   rsync -av --delete "/Users/koishi810/Desktop/韻辞典/曲/" "data/曲/"
   rsync -av --delete "/Users/koishi810/Desktop/韻辞典/分類/" "data/分類/"
   rsync -av --delete "/Users/koishi810/Desktop/韻辞典/アーティスト/" "data/アーティスト/"
   ```

4. 同步完成后运行 `npm run build` 确认构建无误。

5. 若构建通过，执行 git commit + push，触发 GitHub Actions 自动部署。

## 注意
- 删除操作不可逆，步骤2的确认非常重要
- 如果只想同步新增/修改而不删除，去掉 rsync 的 `--delete` 参数
