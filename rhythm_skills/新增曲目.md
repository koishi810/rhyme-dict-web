# 新增曲目

在 `data/曲/` 目录下新建一首歌的 Markdown 条目文件。

## 步骤

1. 根据用户提供的信息，确定以下字段：
   - artist（艺人名）
   - title（曲名）
   - year（发行年）
   - genre（HH / JP-POP / R&B / ROCK 等）
   - cluster_name（9章分类：①同位置韻 / ②同方法韻 / ③清濁韻 / ④連韻 / ⑤広狭韻 / ⑥音色韻 / ⑦重ね韻 / ⑧特殊モーラ韻 / ⑨越境韻）

2. 读取一首已有的曲目文件（如 `data/曲/Creepy Nuts - サントラ.md`）作为格式参考。

3. 在 `data/曲/` 下创建新文件，文件名格式为 `{Artist} - {Title}.md`。

4. YAML frontmatter 必须包含：
   - code、artist、title、year、genre
   - cluster_name、primary_cluster（1-9的数字）
   - existence_status: "not_checked"
   - confidence: "unknown"
   - rhyme_patterns: []

5. 正文包含：
   - `## 歌詞` 节（留空或填入用户提供的歌词）
   - `## 韻分析メモ` 节（留空）

6. 同步到 `data/アーティスト/{Artist}.md`：若艺人文件已存在，在曲目表格中追加一行；若不存在，提示用户需要新建艺人文件。

完成后告知用户文件路径，并提醒运行 `npm run build` 验证页面生成是否正常。
