# 查找相似曲目

根据指定曲目或母音骨格，找出语料库中模式相似的其他曲目。

## 使用方式

用户可以提供以下任一输入：
- 曲名（如「サントラ」「のびしろ」）
- 母音骨格字符串（如 `a-i-o-o`）
- 章分类（如「⑤広狭韻」）

## 步骤

1. 确定目标曲目的特征：
   - 读取 `data/曲/{目标曲}.md` 的 frontmatter
   - 提取 vowel_skeleton、cluster_name、rhyme_patterns

2. 在所有曲目中查找相似项，按以下优先级：
   - **高度相似**：cluster_name 相同 + vowel_skeleton 有2个以上共同子串
   - **同章相似**：cluster_name 相同
   - **母音相似**：rhyme_patterns 有重叠

3. 输出相似曲目列表，每条注明：
   - 艺人 / 曲名 / 年代
   - 相同的母音骨格或韻模式
   - 相似原因（一句话说明）

4. 可选：列出这批相似曲目的共同音韻特征，帮助用户理解这个风格的规律。
