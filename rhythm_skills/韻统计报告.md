# 韻统计报告

读取 `data/曲/` 下所有曲目文件，生成当前语料库的统计摘要。

## 需要统计的内容

### 基础数量
- 总曲目数
- 有完整歌词的曲数（lyrics 非空）
- 有韻グループ分析的曲数（rhymeGroups 数量 > 0）

### 分类分布
- 每个 cluster_name（9章）各有多少曲，按数量排序
- 饼图数据（百分比）

### 艺人分布
- 收录曲数 Top 10 艺人

### 年代分布
- 按年代（1990s / 2000s / 2010s / 2020s）统计曲数

### ジャンル分布
- HH / JP-POP / R&B / ROCK 等各类型数量

### 数据质量
- existence_status 为 verified_spotify 的曲数
- existence_status 为 not_checked 的曲数
- confidence 为 high / medium / low 各多少

## 输出格式
用 Markdown 表格和简洁文字输出报告，方便直接复制到 Obsidian 或文档中。
